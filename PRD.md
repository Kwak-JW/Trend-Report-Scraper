# 트렌드 리포트 자동 수집기 (Trend Report Auto-Scraper) - PRD (v1.0)

## 1. 개요 (Product Overview)
본 제품은 다양한 웹사이트에 분산되어 있는 '트렌드 리포트'를 자동으로 감지하여 사용자의 로컬 PC에 원본 또는 PDF 형태로 자동 저장해 주는 웹 기반 자동화 애플리케이션입니다.

## 2. 목적 (Objectives)
- **시간 절약:** 매번 여러 사이트를 방문해 리포트를 확인하고 다운로드하는 반복 업무를 제거.
- **범용적 접근:** 특정 사이트의 HTML 구조(Class 이름 등)에 종속되지 않고 'Heuristic(경험적 추론)' 기반으로 날짜, 링크, 첨부파일을 감지하여 다양한 사이트에 일괄적으로 대응 가능.

## 3. 핵심 기능 (Core Features)

### 3.1. 사용자 인터페이스 (UI)
- **타겟 도메인 관리:** 스크래핑할 대상 웹사이트 URL을 사용자가 손쉽게 추가/수정/삭제 가능 (기본 제공 프리셋 포함).
- **기간 설정:** 다운로드할 리포트의 기간(시작일~종료일)을 캘린더 컴포넌트를 통해 직관적으로 설정.
- **저장 경로 지정:** 리포트가 저장될 로컬 PC 폴더 경로 지정 (클라우드 환경 구동 시 ZIP 형태 다운로드 지원).
- **실시간 관제 (작업 모니터링):** 
  - SSE(Server-Sent Events)를 통해 UI에 현재 처리 중인 사이트, 진행률, 성공/실패 여부를 실시간 출력.
  - 모든 작업 완료 후 안내 메시지 표기.

### 3.2. 지능형 스크래핑 엔진 (Heuristic Engine)
- **날짜 및 Pagination 감지:** 
  - 화면 상의 `YYYY-MM-DD`, `YYYY년 MM월` 등 날짜 텍스트를 파싱하여 대상 게시물을 식별.
  - 페이징 구조 또는 아코디언 형태의 동적 폼 (`select` 기반)을 자동 추론 및 탐색.
- **문서 및 URL 탐지:** 
  - 리스트에서 가장 긴 텍스트의 `<a>` 태그를 본문 이동 링크로 추출.
  - `onclick` 내의 다양한 인라인 스크립트(`filedownload`, `fileDown`, `window.open` 등) 다운로드 명령 분석 및 직접 다운로드 처리.
  - 단순 문서 뷰어(미리보기, view.jsp 등)는 다운로드 대상으로 오인하지 않도록 사전 필터링.
- **비상 PDF 변환 (Smart Fall-back):** 
  - 명시된 다운로드 파일이 없거나, 다운로드된 대상이 단순 이미지, 텍스트 형태일 경우, 브라우저 렌더링 화면(본문 텍스트 + 이미지)을 자체적으로 병합하여 고품질의 `.pdf` 문서로 강제 변환하여 저장.
- **스마트 네이밍 & 타입 검증:**
  - 리포트 식별을 높이기 위해 난수 파일명 대신 추출한 '리포트 제목'을 기준으로 파일명 생성 (특수문자만 올바르게 치환 처리).
  - 응답 HTTP Header의 `Content-Disposition` 포맷을 평가해 HWP, PDF, ZIP 등 정확한 확장자를 보존.

## 4. 기술 스택 (Tech Stack)
- **Frontend:** React, Tailwind CSS, Vite (실시간 모니터링 및 제어 대시보드)
- **Backend:** Node.js (Express), SSE(Server-Sent Events)
- **Crawling/Scraping:** Puppeteer (Headless 렌더링), Cheerio (DOM 파싱), Axios (스트림 다운로드)
- **Utilities:** date-fns, archiver (ZIP 압축 기능)
