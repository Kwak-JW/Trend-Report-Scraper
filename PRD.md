# [PRD] Trend Report Auto-Scraper (v1.0)

## 1. 제품 개요 (Product Overview)
본 제품은 국내외 주요 경제 연구기관, 글로벌 통상 허브, 공공 정책 진흥기관 및 민간 경제 싱크탱크 등의 전문적인 분석·통상 리포트를 단일 대시보드에서 일괄 수집하는 **지능형 다채널 리포트 수집 플랫폼**이다.

사용자가 탐색하고자 하는 **시작일~종료일** 범위에 정확히 부합하는 핵심 리포트 목록을 감지·식별하고 수집하며, 최종 수집된 보고서 정보와 원본 첨부파일 및 정밀 변환된 PDF 본문을 단일 ZIP 아카이브 파일로 자동 제공한다.

---

## 2. 제품 핵심 가치 및 목표 (Core Product Value & Goals)
- **정보 수집의 자동화 및 생산성 향상**: 산재한 다중 연구 기관의 통상 리포트를 단일 UI 화면에서 동시에 대량 수집할 수 있도록 돕는다.
- **포괄적 수집 복원력**: 각 수집 대상 웹사이트의 설계 특성에 기초한 비표준 다운로드 양식(인라인 스크립트 기반 동작, 프레임 내 팝업 등)과 네트워크 접근 지연 현상을 Heuristic 엔진으로 극복하여 수집 누락을 차단한다.
- **대역폭 비용 및 호스트 가부하 방지**: 사용자 입력을 이용한 원격 서버 측 기간 제한 검색 파라미터 연동, 유효 데이터 유실 페이지 도달 시 탐색 루프 자동 종료 기술을 연계하여 트래픽 낭비와 불필요한 공전을 제거한다.
- **최상급 문서 독해 가독성**: 본문 외 노이즈(GNB, 사이드바, 소셜 배너, 카테고리 정보 등) 구성 요소를 완벽히 제거한 가독성 높은 레이아웃 본문 PDF를 구축해 문서 요약 및 비즈니스 분석 성과에 즉각 기여한다.

## 3. 핵심 기능 요구사항 (Functional Requirements)

### 3.1. 지능형 다중 채널 수집 제어
- **지원 채널**: 다양한 대외 통상 허브 및 공공·민간 정책 산업 연구 유관 기관 통합 지원.
- **지능형 검색 및 탐색 최적화**: 수집 목표 사이트에서 날짜 필터를 구동할 수 있는 시작/종료 파라미터(`pStartDt`, `pEndDt` 등)를 식별하여 서버 수준의 목록 필터링을 우선 적용하고, 순회 도중 더 이상 일치하는 데이터가 발각되지 않으면 즉각 목록 조회를 안심 중단(Paging Terminate Heuristic)한다.
- **무결성 앵커 및 텍스트 블랙리스트 필터**: 헤더 및 푸터 영역, 공용 사이트 디자인에서 유출되는 메타 링크("달력 보기", "지역 정보", "지적 재산관" 등)를 주소 무결성 확보 기술과 블랙리스트 키워드 거부 기법을 사용해 목록 유입을 엄격히 차절한다.

### 3.2. 정밀 상세 수집 및 파일 격리 보존
- **작업 인스턴스별 다운로드 디렉토리 격리**: 동시다발적으로 진행되는 다운로드 수집 트랜잭션마다 고유 세션을 안전하게 부여하고 실시간 완료 데이터 리스트를 격리 추적(Isolation Write List)해 최종 정확한 ZIP 패키지로 합성 이행한다.
- **범용 인버티드 JS 추적**: 브라우저 외부 혹은 클라이언트 인라인 자바스크립트로 실행되는 특수 다운로드 이벤트를 실시간 캡처 및 실행하여 온전한 원본 첨부 문서를 추출한다.

### 3.3. 실시간 관제 및 제어 대시보드 UI
- **실시간 데이터 스트리밍(SSE)**: 세션별 데이터 수집 상황과 디테일한 에러 감지 내역을 UI 패널에 실시간 텍스트 스트리밍 로그로 전송하여 시각적인 관제 경험을 제공한다.
- **비상 긴급 중단 (STOP 기능)**: 수집 도중 사용자가 STOP 버튼을 트리거하면 탐색 중인 비동기 루프 유닛을 안전 종료하며 브라우저 실행 가상 기기의 메모리 및 네트워크 리소스를 일괄 환수한다.
- **활성 대상 사이트 토글**: 활성화하고 싶은 타겟 사이트를 사용자가 원버튼 클릭으로 정밀 제어할 수 있는 뱃지 UI를 제공한다.

---

## 4. 비기능적 요구사항 (Non-Functional Requirements)
- **우회 복원력 및 성능**: 다 채널 요청 집중 분산 기술(`CONCURRENCY_LIMIT=2`), 정상 데스크톱 위장 브라우저 탐색 설정(User-Agent 모사 등) 수립을 바탕으로 웹 탐색 과정에서의 안전성을 확보한다.
- **사용자 경험(UX)**: Tailwind CSS 스타일링을 도입한 고대비 및 눈의 무리가 적은 우아한 대시보드를 배치하여 모니터링 프로세스를 고수 수준으로 장식한다.
- **시스템 신뢰성**: 단기적 네트워크 단절 상태 또는 접속 지연 상황 시 자동 3회 재도전 메커니즘을 이행하여, 안정적이며 일관적인 다운로드 작업 종결을 약속한다.
