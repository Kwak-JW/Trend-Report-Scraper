# Trend Report Auto-Scraper (v1.0)

국내외 주요 정책 연구소 및 대외 경제 분석 기관의 비즈니스·통상 보고서를 기간별로 지능 분석하여 한곳에 모아보고, 상세 리포트 원본 파일 및 소독 정제된 본문 PDF를 단 하나의 ZIP 압축 문서함으로 안전 내려받을 수 있는 **지능형 다채널 통상 정보 일괄 수집 시스템**입니다.

---

## 🌟 핵심 제공 기능 (Key Features)

1. **다채널 지능형 수집 통합**
   - 글로벌 정책 연구소, 경제 진흥원, �   - 메인 웹사이트의 헤더나 공통 푸터 바에서 이중 게재되던 "사이트맵 보기", "관련 메뉴 안내" 등 가짜 스크랩 문서의 수집을 주소 무결성 확보 조건 검증 및 키워드 블랙리스트 필터링 기술로 완전 예방.

3. **인쇄본 품질의 본문 PDF 정화 서비스**
   - 실제 보고서가 들어있는 본문 영역을 우선으로 도려내어 레이아웃 세척 수행.
   - GNB 상단 메뉴 바, 하단 카피라이트 푸터, 공유하기, 소셜 네트워크 버튼, 탑 배너 등 **35가지 종류의 불필요 자화 요소를 완벽 청소**.
   - 실제 간행 인쇄물과 같은 조밀한 가독성을 제공하는 본문 중심의 PDF 자동 변환 폴백을 제공하여 데이터 수집 효율성 극대화.

4. **실시간 모니터링 및 긴급 비상 정지 (STOP)**
   - 실시간 양방향 정보 전송 기술인 SSE (Server-Sent Events) 스트리밍 채널을 적용하여 탐색 세부 로그를 가독 패널 화면에 직관적으로 실시간 출력.
   - 크롤링 중 사이트 차단 및 네트워크 지연 시, 현재 구동 중인 가상 브라우징 인스턴스의 자원을 전부 깔끔히 반환하고 즉각 철수 수립하는 안심 중단(STOP) 안전 루프 구비.

## 💻 일반인 사용자를 위한 원클릭 실행 가이드 (추천)

프로그램이나 명령창 사용에 익숙지 않은 사용자분들도 **마우스 더블 클릭**만으로 매우 간편하게 프로그램을 구동하실 수 있습니다.

### [사전 준비 (최초 1회)] 구동 엔진 설치하기
본 프로그램은 자바스크립트 구동 엔진인 **Node.js**를 필요로 합니다.
1. **[Node.js 공식 홈페이지(https://nodejs.org)](https://nodejs.org)** 에 접속합니다.
2. 화면에 크게 표시된 초록색 **LTS (안정 추천 버전)** 단추를 클릭하여 다운로드합니다.
3. 설치 파일을 실행하여 변경 사항 없이 `Next (다음)`를 계속 눌러 설치를 마칩니다.

---

### 🚀 [Windows] 더블 클릭으로 자동 실행하기

1. 다운로드 받으신 시스템 소스코드 ZIP 파일의 **압축을 안전한 폴더에 완전히 해제**합니다. (예: `C:\TrendScraper` 또는 바탕화면)
2. 압축이 해제된 폴더 안에 있는 톱니바퀴 모양의 **`start-windows.bat` (또는 start-windows) 파일**을 찾아 마우스로 **더블 클릭**합니다.
3. 검은색 명령창이 나타나며 자동으로 필요한 부품을 점검/설치(최초 1회)하고 서버 로직을 가동합니다.
4. 잠시 후 **자동으로 인터넷 브라우저 창이 열리며** 수집기 대시보드 화면이 준비됩니다.
   > 💡 **종료하실 때는** 프로그램이 구동 중인 검은색 명령창 우측 상단의 `X` 버튼을 눌러 창을 닫아주시면 깔끔하게 종료됩니다.

---

### 🍏 [Mac / 수동 실행] 터미널을 이용한 명령창 실행 방법
(Mac 사용자 또는 Windows 환경에서 원클릭 실행 스크립트가 원활하게 작동하지 않는 분들을 위한 정석 방법입니다.)

1. 명령창(Mac은 터미널, Windows는 cmd)을 열어 다운로드 받은 폴더의 실제 경로로 진입합니다.
   ```bash
   cd "압축 풀어놓은 폴더의 전체 주소"
   ```

2. 아래 명령어를 한번 치고 엔터를 눌러 작동 부품들을 초기 설치합니다. (최초 1회 실시, 수 분 소요)
   ```bash
   npm install
   ```

3. 설치가 완료되면 아래 기동 명령어를 입력하여 서버를 구동합니다.
   ```bash
   npm run dev
   ```

4. 평소 쓰시는 웹 브라우저(크롬, 웨일, 엣지 등)를 열고 주소창에 굵은 글씨 주소(**`http://localhost:3000`**)를 입력하여 접속하시면 대시보드에 진입합니다.
5. 시각적으로 나타난 관제 화면에서 **분석을 행하고 싶은 날짜 범위(시작/종료일)**를 선택하고, 탐색할 기관 사이트 뱃지들을 조절한 후 **수집 시작** 버튼을 누르면 실시간 관제 연출과 함께 다운로드가 시작되며, 작업 완성 즉시 최상급 인쇄 품질의 ZIP 압축 폴더를 제공받습니다.

## 🛠 전문 개발자용 서비스 배포 및 빌드 명세 (For Developers)

- **Frontend Specs**: React 18 / Tailwind CSS / Lucide React Icons
- **Backend Core**: Express / Puppeteer Browser Module / Cheerio / Axios
- **Production Build (프로덕션 상용 서비스 배포 시)**:
  > 상용 환경으로 배포하기 전, 백엔드와 프론트엔드의 압축 컴파일(빌드)이 반드시 선행되어야 합니다. 빌드 없이 `npm start`를 실행하면 `MODULE_NOT_FOUND (dist/server.cjs)` 에러가 발생합니다.

  ```bash
  npm run build
  npm start
  ```
  *(빌드 프로세스 중 TypeScript 백엔드 서버가 `dist/server.cjs` 압축 자립 서버 단일 코드로 esbuild 번들링되기 때문에 ES Module 버전이나 시스템 파일 I/O 레이턴시에 따른 충돌 없이 안정 구동과 지속 유지를 항구 장담합니다.)*
