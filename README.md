# Trend Report Auto-Scraper (v1.0)

국내외 주요 정책 연구소 및 대외 경제 분석 기관의 비즈니스·통상 보고서를 기간별로 지능 분석하여 한곳에 모아보고, 상세 리포트 원본 파일 및 소독 정제된 본문 PDF를 단 하나의 ZIP 압축 문서함으로 안전 내려받을 수 있는 **지능형 다채널 통상 정보 일괄 수집 시스템**입니다.

---

## 🌟 핵심 제공 기능 (Key Features)

1. **다채널 지능형 수집 통합**
   - 글로벌 정책 연구소, 경제 진흥원, 국무총리 산하 싱크탱크 및 민간 경제 전문 연구 공간 연동 지원.
   - 무조건적인 리스트 순회 대신 사용자가 설정한 수집 최소/최대 기간 매개변수를 검색 필터로 유기적으로 조율 반영.

2. **지능형 순회 회피 및 자원 정밀 제어 (Smart Termination)**
   - 좁은 범위의 지정 단일 날짜 탐색 시, 매진 구간 이후에도 무의미하게 다음 페이지들을 낭비 조회하지 않도록 돕는 **목록 조기 영리 정지(Paging Terminate Heuristic)** 기능 내장.
   - 메인 웹사이트의 헤더나 공통 푸터 바에서 이중 게재되던 "사이트맵 보기", "관련 메뉴 안내" 등 가짜 스크랩 문서의 수집을 주소 무결성 확보 조건 검증 및 키워드 블랙리스트 필터링 기술로 완전 예방.

3. **인쇄본 품질의 본문 PDF 정화 서비스**
   - 실제 보고서가 들어있는 본문 영역을 우선으로 도려내어 레이아웃 세척 수행.
   - GNB 상단 메뉴 바, 하단 카피라이트 푸터, 공유하기, 소셜 네트워크 버튼, 탑 배너 등 **35가지 종류의 불필요 자화 요소를 완벽 청소**.
   - 실제 간행 인쇄물과 같은 조밀한 가독성을 제공하는 본문 중심의 PDF 자동 변환 폴백을 제공하여 데이터 수집 효율성 극대화.

4. **실시간 모니터링 및 긴급 비상 정지 (STOP)**
   - 실시간 양방향 정보 전송 기술인 SSE (Server-Sent Events) 스트리밍 채널을 적용하여 탐색 세부 로그를 가독 패널 화면에 직관적으로 실시간 출력.
   - 크롤링 중 사이트 차단 및 네트워크 지연 시, 현재 구동 중인 가상 브라우징 인스턴스의 자원을 전부 깔끔히 반환하고 즉각 철수 수립하는 안심 중단(STOP) 안전 루프 구비.

## 💻 일반인 사용자를 위한 설치 및 실행 가이드

프로그램이나 컴퓨터 명령창 사용에 익숙지 않은 일반 사용자분들도 아래 기동 단계를 차근차근 따라 하시면 간편하게 본 프로그램을 구동하여 사용하실 수 있습니다.

### [사전 준비] 내 컴퓨터에 구동 엔진 다운로드하기
본 프로그램은 자바스크립트 가상 작동 엔진인 **Node.js**를 필요로 합니다. 아래 절차에 따라 간편하게 가상 엔진을 구축해 주세요.

1. 인터넷 브라우저 주소창에 **[Node.js 공식 홈페이지](https://nodejs.org)** 주소를 입력하여 접속합니다.
2. 메인 화면에 크게 표시된 초록색 **LTS (안정 추천 버전)** 단추를 클릭하여 설치 프로그램을 다운로드합니다.
3. 다운로드 완료 후 설치 파일을 실행하여 진행 중인 모든 `Next (다음)` 안내 단추를 선택 완료하여 컴퓨터 설치를 마칩니다.

---

### [Step 1] 프로그램 소스코드 전체를 컴퓨터로 가져오기
가장 확실하고 쉬운 다운로드 방식인 압축 파일 압축 풀기 방식을 권장합니다.

1. 전달받은 프로그램 전체 소스코드가 동봉된 ZIP 압축 파일을 준비합니다.
2. 해당 ZIP 파일을 컴퓨터의 안정적인 디렉토리 경로(예: `C:\TrendScraper` 또는 맥의 `바탕화면`)에 옮긴 뒤 **압축을 완전히 해제**합니다.

---

### [Step 2] 윈도우/맥 명령창(터미널) 열어 진입하기

* **윈도우(Windows) 컴퓨터 사용자**:
  1. 키보드의 `[시작 키 (또는 Ctrl 옆의 윈도우 로고 키)]`를 누릅니다.
  2. 단어 **cmd** 또는 **명령 프롬프트**를 자판으로 검색한 뒤 마우스로 눌러 검은색 명령제어 창을 실행합니다.
  3. 압축을 해제했던 실제 경로 폴더로 뛰어들기 위해 아래 명령어를 양식에 맞추어 타이핑해 넣고 키보드 `[Enter]`를 침니다.
     ```cmd
     cd "압축을 풀어놓으신 폴더의 실제 전체 주소"
     (예시): cd C:\Users\Username\Desktop\TrendScraper
     ```

* **맥(macOS) 컴퓨터 사용자**:
  1. 키보드의 `[Command + SpaceBar]` 단축키를 눌러 Spotlight 검색창을 켭니다.
  2. 단어 **터미널 (Terminal)**을 타이핑해 엔터를 쳐 실행합니다.
  3. 아래 명령어를 이용해 경로에 접근합니다.
     ```bash
     cd "압축을 해제하신 폴더의 전체 주소"
     (예시): cd ~/Desktop/TrendScraper
     ```

---

### [Step 3] 필요한 세부 패키지 구성 부품 자동 설치하기
처음 한 번만 가동해 주는 프로그램 작동 부품 패키지 일괄 설치 과정입니다. 명령창에 아래 한 줄을 그대로 써넣은 후 키보드 `[Enter]`를 누릅니다.

```bash
npm install
```
*(인터넷 환경에 맞추어 프로그램 전산 분석에 꼭 필요한 수집 전용 가상 장비와 디자이너 도구를 안전하게 자동 배치하기 시작합니다. 단 몇 초에서 몇 분 정도 기다리시면 조용하게 준비가 성공 완료됩니다.)*

---

### [Step 4] 수집기 대시보드 기동 및 사용하기
모든 예비 준비가 완료되어 가동할 마지막 단계입니다. 아래 기동 시작 키워드를 넣고 `[Enter]`를 칩니다.

```bash
npm run dev
```

1. 명령창 터미널 줄에 아래와 같은 초록 메시지가 밝게 활성화되면 정상 기동 상태입니다.
   `Server running on http://localhost:3000`
2. 즐겨 쓰시는 인터넷 웹 브라우저(크롬, 웨일, 에지 등)의 상단 주소 필드에 접속 주소를 입력해 이동합니다.
   > **접속 링크 주소**: `http://localhost:3000`
3. 시각적으로 나타난 관제 화면에서 **분석을 행하고 싶은 날짜 범위(시작/종료일)**를 선택하고, 탐색할 기관 사이트 뱃지들을 조절 토글한 후 **수집 시작** 버튼을 누르면 실시간 관제 연출과 함께 다운로드가 시작되며, 작업 완성 즉시 최상급 인쇄 품질의 ZIP 압축 폴더를 수령하여 편리하게 정비된 최신 세부 리스를 누릴 수 있습니다!

## 🛠 전문 개발자용 서비스 배포 및 빌드 명세 (For Developers)

- **Frontend Specs**: React 18 / Tailwind CSS / Lucide React Icons
- **Backend Core**: Express / Puppeteer Browser Module / Cheerio / Axios
- **Production Build (프로덕션 상용 서비스 배포 시)**:
  ```bash
  npm run build
  npm start
  ```
  *(빌드 프로세스 중 TypeScript 백엔드 서버가 `dist/server.cjs` 압축 자립 서버 단일 코드로 esbuild 번들링되기 때문에 ES Module 버전이나 시스템 파일 I/O 레이턴시에 따른 충돌 없이 안정 구동과 지속 유지를 항구 장담합니다.)*
