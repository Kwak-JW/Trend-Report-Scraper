# Trend Report Auto-Scraper v1.0

이 프로젝트는 지정된 웹사이트들을 순회하여 기간 내 작성된 트렌드 리포트를 자동으로 색인하고 로컬 환경에 원본 리포트 또는 PDF 형태의 리포트로 추출해주는 강력한 범용 스크래핑 도구입니다.

## ⚠️ 클라우드 환경 vs 로컬 환경
이 코드는 AI Studio(클라우드 샌드박스) 환경 및 로컬 PC 환경 양쪽에서 모두 안전하게 작동하도록 풀스택(Full-stack) 아키텍처로 구현되었습니다.
- **클라우드 구동 시:** 다운로드 파일은 1차적으로 서버(컨테이너)의 지정 경로에 임시 저장되며, 작업 종료 후 브라우저상에서 하나의 `.zip` 압축 파일로 일괄 다운로드됩니다.
- **로컬 구동 시 (권장):** 사용자가 대시보드에서 지정한 `C:\downloads\reports` 등 **자신의 실제 로컬 경로**에 수집된 원본 파일 및 변환된 PDF가 직접 저장되므로 즉각적인 확인이 가능합니다.

## 🚀 로컬 PC에서 실행하는 방법 (단계별 안내)

로컬 PC에서 가장 완벽하게 작동합니다. 다음 단계를 순서대로 진행해 주세요.

### 1단계: 필수 소프트웨어 설치
PC에 **Node.js**가 설치되어 있어야 합니다. (v18 이상 권장)

### 2단계: 필수 패키지 설치
1. 다운로드 받은 프로젝트 폴더에서 터미널(명령 프롬프트, PowerShell, Mac Terminal 등)을 엽니다.
2. 아래 명령어를 입력하여 크롤링과 서버 구동에 필요한 모든 라이브러리를 설치합니다.
   ```bash
   npm install
   ```
   *(주요 백엔드 기술 스택: `Express`, `Puppeteer`, `Cheerio`, `Archiver`, `Axios` 등)*

### 3단계: 애플리케이션 서버 실행
패키지 설치가 완료되었다면, 터미널에 다음 명령어를 입력합니다. (프론트엔드와 백엔드를 동시에 실행)

```bash
npm run dev
# 또는
npm start
```
`Server running on http://localhost:3000` 문구가 출력되면 정상동작 상태입니다.

### 4단계: 스크래핑 이용하기
1. 웹 브라우저(Chrome 권장)를 열고 http://localhost:3000 으로 접속합니다.
2. **타겟 도메인 세팅:** 스크래핑하고자 하는 대상 사이트 모음을 구성합니다 (자주 쓰는 기본 예시 제공됨).
3. **탐색 구간:** 캘린더 입력기를 이용해 리포트를 탐색할 구간(시작일과 종료일)을 선택합니다.
4. **저장 폴더 (옵션):** PC에 집어넣을 자신의 OS 형식 로컬 폴더 경로를 기재하면, 그곳으로 즉시 다운로드가 이루어집니다.
5. **자동 스크래핑 시작** 버튼을 누릅니다.
6. 관제 로그(모니터링) 창에서 브라우저가 직접 페이지들을 순회하고 문서들 식별 및 다운로드 받는 일련의 실시간 과정을 확인합니다.

## 💡 v1.0 통합엔진의 동작 원리 (휴리스틱 추론)

v1.0의 통합 크롤러는 각 사이트를 일일이 분석해서 맞춤형 코딩을 할 필요가 없는 '범용 휴리스틱(Heuristic)' 구조로 개발되었습니다.

- **스마트 날짜 파싱:** 웹 화면 상에 텍스트 형태로 노출된 `YYYY-MM-DD`, `YYYY.MM.DD`, `YYYY년 MM월` 등 다양한 규칙들을 자동 감지하여 기간 내 글인지 판단합니다.
- **다차원 문서 및 다운로드 탐지:** 
  - 특정 클래스명에 얽매이지 않고 요소들 중 `가장 텍스트가 긴 링크`를 리포트 상세 페이지로 인식하고 접근합니다.
  - 리스트 내부에 직접 다운로드 버튼이 있는 경우(예: `filedownload`, `fileDown`) 이를 감지하고 인라인 스크립트를 파싱해 다이렉트로 곧바로 파일을 빼옵니다.
  - 미리보기나 문서 웹뷰어 기능을 제공하는 사이트는 정교하게 해당 뷰어 URL 들을 무시하고 오직 '다운로드' 만 진행시킵니다.
- **확장성을 가진 파일 명명:** 무작위 난수 기반의 파일명은 저장 후 구분이 안되는 불편함이 큽니다. 제목을 곧바로 파일명으로 쓰도록 자동 변환하며 꺾쇠, 물음표 등 OS에서 허용하지 않는 특수기호만 걸러냅니다. 
- **비상용(Fall-back) PDF 캡처 기능 (Smart PDF Conversion):** 
  대상 리포트가 PDF 형태로 첨부를 올려놓지 않았거나, 본문 텍스트 내에 내용들을 나열만 해둔 경우 또는 단순 이미지 텍스트 파일인 경우(거짓 양성 판별), 엔진은 이를 인식하여 즉시 Puppeteer 브라우저 렌더링 화면을 스냅하여 단일 문서를 가진 `.pdf` 형태로 강제 변환하여 드롭해줍니다.
